Promises

Objeto usado como placeholder de eventuales resultados.

Representa un evento que va a ocurrir, pero no AHORA (asincronismo).

Posibles Estados: fulfilled, rejected o pending.

Pueden encadenarse then() & catch()

Maps

Estructura de datos tipo diccionario que permite cualquier valor como key.

WeakMaps misma API pero no se puede iterar sobre el contenido

Sets

Estructura de datos tipo tipo conjunto, donde los valores son unicos.

WeakSets misma API pero no se puede iterar sobre el contenido

Iterator & for..of Loop

Objeto con metodo next() que devuelve una tupla { code, value }

String, Array, TypedArray, Map and Set son iterables.
